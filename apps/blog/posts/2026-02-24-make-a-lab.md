# 실험실 만들기

1. 인프라와 배포

- 터보레포 사용 모노레포 환경에서 패키지 구조화 및 pnpm --filter를 활용한 빌드 최적화
- vercel 배포 트러블슈팅
  - 경로 이슈(루트, 아웃풋)

2. 프론트엔드 작업

- slug 사용 동적 라우팅 및 SSG
- Contentlayer로 마크다운 콘텐츠를 JSON 데이터로 변환
- ESM \_\_dirname 대신 fileURLToPath를 사용하여 Node.js 환경에서의 경로 참조 문제 해결

3. 최적화

- 커밋 규칙
- git ignore

4. AI 활용

- AI Agent Workflow: 블로그 포스팅 제목과 슬러그 자동생성
