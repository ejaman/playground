웹과 임베디드 환경(script 태그 삽입)을 동시 지원하는 통계 시각화 모듈 1인 개발. 콘텐츠 제작자가 일반 객체를 다루듯 데이터를 제어할 수 있도록 추상화 계층을 설계했습니다.

## 담당 파트

- 데이터 공유 구조 설계 (Proxy 기반 Redux 양방향 바인딩)
- D3.js 기반 통계 그래프 컴포넌트 구현
- 표 컴포넌트 설계 및 구현
- UI 기획 (PPT 기획안 작성)

## 핵심 도전

**Proxy 기반 Redux 추상화**: 콘텐츠 제작자에게 Redux의 복잡한 액션 dispatch를 노출하지 않기 위해 Proxy setter가 일반 객체 할당을 가로채 Redux 액션을 자동으로 dispatch하는 구조를 구현

**D3.js 맞춤형 그래프**: 기성 차트 라이브러리는 세밀한 커스텀이 불가하고 그래프 추가마다 의존성이 증가함. D3.js로 직접 구현해 join 패턴으로 DOM 조작을 최적화하고 transition으로 인터랙션을 구현.

## 배운 것

아토믹 디자인의 실질적 이점을 체감한 프로젝트로 컴포넌트 단위로 디자이너와 협업, 새로운 표가 추가될 때 기존 컴포넌트를 조합해 바로 구현 가능.
